## 📌 Project Overview

The **Student Ranking System** is a Flask-based web application that manages student records and calculates student rankings using the **Heap Sort Algorithm**. The system provides an efficient way to store marks, generate rankings, and display results through an interactive web interface.

---

## 🎯 Objectives

* Manage student academic records efficiently.
* Calculate rankings using Heap Sort.
* Demonstrate sorting algorithm implementation in a real-world application.
* Provide a simple and user-friendly dashboard.

---

## ✨ Features

✅ Add Student Records

✅ Store Student Marks

✅ Calculate Rankings using Heap Sort

✅ Display Ranked Student List

✅ Interactive Dashboard

## 🧠 Algorithm Used

### Heap Sort

Heap Sort is used to generate student rankings efficiently.

### Time Complexity

| Case    | Complexity |
| ------- | ---------- |
| Best    | O(n log n) |
| Average | O(n log n) |
| Worst   | O(n log n) |

Advantages:

* Fast sorting performance
* Efficient ranking generation
* Suitable for large datasets
